# Security Policy

WebHachimi is an early-stage browser game editor/runtime project. Please report
security issues privately before opening a public issue.

## Reporting a Vulnerability

Email the maintainer at the address associated with the GitHub profile, or open
a GitHub issue with only a high-level description and request a private contact
channel. Please do not include exploit details in a public issue.

Include:

- Affected component or URL.
- Steps to reproduce.
- Impact and severity estimate.
- Suggested fix, if known.

The maintainer will acknowledge reports as soon as practical and coordinate a
fix or mitigation before public disclosure.
