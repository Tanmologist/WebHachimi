# Code of Conduct

WebHachimi aims to be a welcoming project for people learning, building, and
maintaining browser game tools.

## Expected Behavior

- Be respectful and constructive.
- Focus feedback on code, design, reproducibility, and project goals.
- Assume good intent, but be clear when behavior or changes create risk.
- Keep discussions accessible to contributors with different experience levels.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Dismissive comments about someone's background, identity, or ability.
- Publishing private information without permission.
- Repeated disruption of issues, pull requests, or maintainer workflows.

## Reporting

Report conduct concerns privately to the maintainer through GitHub profile
contact channels. Maintainers may remove comments, close issues, or restrict
participation to keep the project healthy.
